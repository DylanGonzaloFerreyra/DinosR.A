# DinosR.A 🦕

**Augmented Reality Educational App**

Mobile AR application built with Unity and Vuforia SDK that detects image targets to render 3D models with multimedia events.

---

## Download

Head to the [**Releases**](../../releases/latest) section to download the latest APK and install it directly on your Android device.

> Requires Android 8.0 (Oreo) or higher.

---

## How It Works

1. Download and install the APK on your Android device.
2. Open the app and point your camera at any of the supported image targets (book covers, posters, etc.).
3. Watch 3D dinosaur models and multimedia content come to life in augmented reality.

---

## Features

- Image-target detection powered by Vuforia Engine
- 3D dinosaur and skeleton models rendered in real time
- Multiple target support (books, posters, and more)
- Optimized for Android mobile devices

---

## Built With

- [Unity 6](https://unity.com/) — game engine and build pipeline
- [Vuforia Engine 11](https://developer.vuforia.com/) — AR image tracking SDK
- C# — scripting
- Android / IL2CPP — target platform and scripting backend

---

## Educational Reference

This app was built around the book **"Tiranosaurio Rex — El increíble mundo de los dinosaurios"**, used as an educational reference for the 3D animations and as the physical base for demonstrating augmented reality use cases. The book's pages and covers serve as image targets, bringing the content to life when viewed through the app.
